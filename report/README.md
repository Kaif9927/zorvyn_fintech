# Project Report (LaTeX) — compact (~15 pages)

`report.tex` is a **short** academic report for a **personal finance management system** (full-stack: React + Express + MySQL + Prisma). It is written to stay **around 15 pages or fewer** when compiled (11pt, 2cm margins).

## Compile

```bash
cd report
pdflatex report.tex
pdflatex report.tex
```

Or upload `report.tex` to [Overleaf](https://www.overleaf.com) and click **Recompile**.

## Front matter

The PDF starts directly with **Section 1 (Introduction)**. There is no separate title page, certificate, declaration, abstract, or table of contents (those were removed on request).
